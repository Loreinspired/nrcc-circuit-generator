# NRCC Circuit · Monthly Generator

**New Reality Christian Centre — ADO-EKITI Campus**  
Power, Light & Sound Department — Monthly Circuit Budget & Report Generator

## Features

- 7-step wizard: Month Setup → Weekly Events → Team & Report → Roster → Performance → Spiritual Activities → Generate
- Auto-calculates Diesel & Petrol fuel costs per week and per service
- Roster & role assignments (GEN, MIC, HVAC, SND, REC, EXT, COORD)
- Steward performance tracking with monthly score cards
- Spiritual activities log (Friday meetings + Saturday prayer)
- 📋 Service Log drawer — log attendance, equipment notes per service
- 🔒 Admin-protected view with summary dashboard
- Download Budget as Excel + Monthly Report as Word
- localStorage persistence — data survives page refreshes

## Usage

Open `index.html` directly in any modern browser (Chrome, Edge, Firefox).  
No server required — everything runs client-side.

## Admin Access

The admin password is set at first use. Contact your Circuit department head for credentials.
